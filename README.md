# 终端DDZ
## 构建服务端
```go
go build -o server main.go
```
## 构建客户端
```go
go build -o cli client/cli.go
```
## 运行服务端
服务端监听 32123 端口
```shell
./server
```
## 运行客户端
客户端连接后需要等待，直至有3人加入游戏后，游戏自动开始
```shell
./cli -u "你的用户名" -h "服务端IP地址"
```
